### &#128580; 충돌없이 PUSH 하는 방법!!!!
------------------------------------------
1. git add .
2. git stash
3. git status
4. git pull origin master
5. git stash pop
6. git add .
7. git commit -m ' '
8. git push origin master

&#128549; 진짜 가끔 commit까지 해서 push 하려는데 그전에 다른사람이 push를 해서 에러가 뜨는경우!!
1. git reset --soft HEAD^
2. git stash ///위의 2번과정부터 다시시작!
